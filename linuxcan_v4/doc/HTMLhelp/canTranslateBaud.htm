canTranslateBaud

Syntax


#include <canlib.h>
canStatus canTranslateBaud(
    long *freq, int *tseg1, 
    int *tseg2, int *sjw,
    int *nosamp, int *syncMode);

Description

This function translates the BAUD_xxx constants to their corresponding bus parameter values.

TSeg1 is the number of quanta (less one) in a bit before the sampling point. TSeg2 is the number of quanta after the sampling point.

Input/Output Parameters

freq
A pointer to a DWORD which contains the BAUD_xxx constant to translate. At return, this DWORD contains the actual bit rate (in bits per second).
tseg1
A pointer to a buffer which receives the Time segment 1, that is, the number of quanta from (but not including) the Sync Segment to the sampling point.
tseg2
A pointer to a buffer which receives the Time segment 2, that is, the number of quanta from the sampling point to the end of the bit.
sjw
A pointer to a buffer which receives the Syncronization Jump Width; can be 1,2,3, or 4.
noSamp
A pointer to a buffer which receives the number of sampling points; can be 1 or 3.
syncmode
Unsupported, always read as zero.
Return Value

canOK (zero) if success
canERR_xxx (negative) if failure.
